# 프로젝트
### 요약
* 설명 : 2018년 국비지원 웹 개발자 양성과정에서 진행했던 파이널 프로젝트(팀)
* 기간 : 약 한달(2018.7)
* 주제 : 반려동물을 임시 보호하고 케어해주는 펫시팅 중개 사이트
* 사용 기술 : 
  - JAVA(JSP / Servlet - MVC2) 
  - DB : ORACLE 11G
  - HTML / CSS
  - JavaScript / jQuery
  - Spring FrameWork(5.0.6)
  - Tomcat 8.5
* **담당 부분** : 
  - 팀장
  - 게시판
